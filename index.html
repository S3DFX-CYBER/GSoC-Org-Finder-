About the Program

What is Google Summer of Code?

Google Summer of Code (GSoC) is an annual global program sponsored by Google that introduces contributors to open source software development. Since 2005, GSoC has brought together 19,000+ participants from 112 countries with 800+ open source organizations.

Contributors work on real-world coding projects under the guidance of experienced mentors, gain valuable skills, build professional networks, and earn a stipend — all while making meaningful contributions to projects used by millions.
